# Documentation

Bu klasör, repoda takip edilen teknik dokümantasyon dosyalarını toplar. Amaç; kök dizini sade tutmak ve altyapı ile uygulama dokümanlarını belirli başlıklara ayırmaktır.

## İçerik

- [infra/](./infra/)
  AWS, deployment ve altyapı notları
- [app/](./app/)
  Uygulama katmanları ve servis yapısına dair giriş dokümanları

## Takip Edilmeyen Lokal Notlar

Ders raporu, rollout kayıtları ve benzeri kişisel çalışma dosyaları bu klasörde tutulmaz. Bu tür dosyalar repoda izlenmeyen `docs-local/` yapısında saklanır.
